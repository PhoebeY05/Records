Formatting Notes

  • Completed: title plus finish date in brackets, for example: Harry Potter and the Philosopher's Stone（17 Jan 2019 end)
  • To Be Read: supports the same metadata markers (for example: #nice, ✔, series marker like }98).
  • Unfinished: title with unread marker, for example: Harry Potter and the Philosopher's Stone <Chapter 120>.
  • Duplicate handling: Completed increments reread, To Be Read keeps earliest in file, Unfinished keeps latest in file.
