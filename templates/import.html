Formatting Notes

  • Completed: title plus finish date in brackets, for example: Harry Potter and the Philosopher's Stone（17 Jan 2019 end)
  • To Be Read:
    • Rich documents (Pages, Word, etc.): the font size of each book line sets its category — largest font = Finished, middle font = Left Extras, smallest font = Uncompleted.
    • Plain text: use section lines, where Finished means finished items, Left Extras or 🙏 means left extras, and plain text means uncompleted.
    • Metadata markers like #nice, ✔, and series markers like }98 still work on book lines.
  • Unfinished: title is everything before the first progress marker, and notes start at that marker. Supported markers include 章, 页, Pg, page, chapter, Chapter, volume, Volume, plus forms like 14章, 第661章, 12页, and 491章/ 77章. The helper text "(haven't read)" is stripped out on import if you include it beside the title.
  • Duplicate handling: Completed adds 1 reread per duplicate and ignores any extra read-count text on the duplicate line, To Be Read keeps earliest in file, Unfinished keeps latest in file.
